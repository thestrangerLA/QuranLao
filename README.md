# QuranLao (ຣໍມະ)

ແອັບພລິເຄຊັນອິດສະລາມໃນພາສາລາວ ທີ່ລວມເອົາຄຳພີອັນກຸຣອານ, ພະນາມຂອງອັລລໍຫ໌ ແລະ ບົດຄວາມຄວາມຮູ້ຕ່າງໆ.

## ຄຸນສົມບັດ (Features)

- **ອັນກຸຣອານ (Quran):** ອ່ານກຸຣອານພ້ອມຄຳແປພາສາລາວ, ອັງກິດ ແລະ ໄທ.
- **ພະນາມຂອງອັລລໍຫ໌ (99 Names of Allah):** ສຶກສາຄວາມໝາຍ ແລະ ລາຍລະອຽດຂອງ 99 ພະນາມອັນຍິ່ງໃຫຍ່.
- **ຮາດີດ (Hadith):** ລວມບົດຮາດີດທີ່ສຳຄັນ ແລະ ຮາດີດນະບະວີ 40 ບົດ (Nawawi 40 Hadith).
- **ບົດຄວາມ (Articles):** ເນື້ອຫາພື້ນຖານສຳລັບຜູ້ທີ່ສົນໃຈອິສລາມ ແລະ ມຸສລິມໃໝ່.
- **ໂໝດມືດ (Dark Mode):** ຮອງຮັບການໃຊ້ງານທັງກາງເວັນ ແລະ ກາງຄືນ.

## ການພັດທະນາ (Development)

ເລີ່ມຕົ້ນ Server ສຳລັບການພັດທະນາ:
```bash
npm run dev
```

## ການອັບເດດລົງ GitHub (Deployment to GitHub Pages)

ເມື່ອທ່ານຕ້ອງການອັບເດດການປ່ຽນແປງລົງໃນເວັບໄຊ GitHub Pages ຂອງທ່ານ, ໃຫ້ໃຊ້ຄຳສັ່ງລຸ່ມນີ້ໃນ Terminal:

```bash
# 1. ກຽມໄຟລ໌ທັງໝົດ
git add .

# 2. ບັນທຶກການປ່ຽນແປງ (ທ່ານສາມາດປ່ຽນຂໍ້ຄວາມໃນ "" ໄດ້)
git commit -m "ອັບເດດຄຳແປອັນກຸຣອານ ແລະ ເນື້ອໃນຮາດີດ"

# 3. ສົ່ງຂຶ້ນ GitHub
git push origin main
```

ລະບົບ GitHub Action ຈະເຮັດວຽກໂດຍອັດຕະໂນມັດເພື່ອ Build ແລະ Deploy ເວັບໄຊຂອງທ່ານໃຫ້ເປັນເວີຊັນລ່າສຸດ. ທ່ານສາມາດຕິດຕາມສະຖານະໄດ້ທີ່ແຖບ "Actions" ໃນ GitHub ຂອງທ່ານ.
